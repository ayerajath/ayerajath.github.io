Cursor and Claude are the future of coding. For programming and developing, I haven't come across better tools than these.

The rate at which I can ship stuff is faster than ever. 

I was able to get this blog up and running in a matter of hours. Earlier, I would have spent days, if not weeks, getting this done. 

If this is not the future, I don't know what is. 

The next few years, there will be more and more tools like these and each one of these will probably be better than the last. The rate at which applications and softwares will be shipped or created will be on a rise. 

How can one stand out? Even I don't know, maybe we'll find out.
What will I be doing? Learn more, Experiment more. Build more. Put a name out there.

Some roles that I feel will be more relevant in the future, are either cracked product engineers or low level engineers. 

This is the way building is headed. A worker always in the IDE for a programmer. Let's see where this goes.
